# Настройка продажи через Plati

## Создание товара

1. Войдите в кабинет продавца Plati и создайте новый цифровой товар.
2. Укажите название: **«7 AI-проектов для школьника на лето — PDF-набор»**.
3. Установите цену **990 ₽**.
4. Добавьте описание товара из раздела ниже.
5. Загрузите финальный PDF или настройте понятную инструкцию получения материала. Для тестового запуска явно укажите, если ручная отправка может занять до 24 часов.
6. Проверьте обложку, формат файла, цену, контакт для поддержки и текст, который покупатель увидит после оплаты.
7. Опубликуйте товар и скопируйте его публичную ссылку.
8. Вставьте ссылку в `SITE_CONFIG.paymentUrl` файла `config.js`.
9. Откройте сайт с тестовой UTM-меткой и проверьте все кнопки покупки в hero, карточке цены и финальном блоке.

Пример настройки:

```js
const SITE_CONFIG = {
  paymentUrl: "https://ваша-ссылка-на-товар.example",
  productPrice: "990 ₽",
  productName: "7 AI-проектов для школьника на лето",
  contactEmail: "support@example.ru"
};
```

Не добавляйте в `config.js` пароль от Plati, токен API или другие секреты. Сайту нужна только публичная ссылка товара.

## Готовое описание товара

**7 AI-проектов для школьника на лето**

Готовый PDF-набор заданий для детей и подростков 10–16 лет, которые хотят попробовать нейросети и создать свои первые AI-проекты.

Это не сложное программирование и не скучная теория. Ребенок шаг за шагом создает понятные результаты: персонажа, мир проекта, постер, мини-бренд, карточку товара, идею короткого видео и финальную презентацию.

Что входит:

— 7 пошаговых заданий  
— готовые промпты  
— идеи для разных возрастов  
— инструкция для родителей  
— правила безопасной работы с ИИ  
— финальное задание для презентации проекта

Это самостоятельный набор заданий, а не живой курс с преподавателем.

## Проверка перед продажами

- цена на Plati и на лендинге совпадает;
- файл открывается и корректно отображает кириллицу;
- покупатель понимает, получит файл автоматически или в течение 24 часов;
- после оплаты показана инструкция получения;
- контактный email работает;
- публичная ссылка товара открывается без входа в кабинет продавца;
- UTM-параметры не мешают открытию страницы оплаты.
